# Modules used in the project

  • vue-router
  • vue-notifyjs (minimalist notification component for Vue 2.x)
  • vee-validate (template-based validation framework for Vue.js that allows you to validate inputs and display errors)
  • lang (element-ui/lib/locale/lang/en)
  • locale (element-ui/lib/locale)
  • vue-js-modal (Modal Component for Vue.js)
  • pouch-vue (pouchdb libs)
  • vue-clickaway (detects clicks outside of the element)
  • sweetalert2 (alert dialogs)
