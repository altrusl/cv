# CouchDB databases

  • users - users of the system. Can be one of three types: admin, subadmin, school user
  • schools - schools. Each school is associated with one school user
  • curriculums - curricullums (courses/classes). Example: GS | Mathematik | Jahrgang 3 / 4 | Raum & Form
  • lessons - topics. Example: GS | Mathematik | Jahrgang 3 / 4 | Zeichnen 3/4
