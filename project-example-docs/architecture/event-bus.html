# Event Bus

# SiFC project utilizes Event Bus design pattern

An EventBus is a mechanism that allows different components to communicate with each other without knowing about each other. A component can send an Event to the EventBus without knowing who will pick it up or how many others will pick it up. Components can also listen to Events on an EventBus, without knowing who sent the Events. That way, components can communicate without depending on each other.

# List of current events:

  • USERS_UPDATED - 'users' database was synchronized with the server

# How to emit event

import EventBus from '@/eventBus'

// ...

EventBus.$emit("USERS_UPDATED");

# How to listen to event

import EventBus from '@/eventBus'

// ...

EventBus.$on('USERS_UPDATED', () => {
    console.log("users are updated");     
});
